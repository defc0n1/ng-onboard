ng-onboard

ng-onboard

AngularJS directives for onboard.js

Demo

Setup

Include the JS and CSS files for onboard.js, and ng-onboard.min.js. Add the module ng-onboardto your app declaration.

Options

Set the options like so: ng-onboard-options="onboardOptions". Define $scope.onboardOptionsin your controller. The format is exactly the same as the original.

Methods

Pick a method name, ng-onboard-method="startOnboard". You can invoke the intro from elsewhere by calling startOnboard();. ng-onboard-stop="stopOnboard". You can invoke the intro from elsewhere by calling stopOnboard();.

See index.html and app.js for usage code.

Example

Set the options and method name: 

<div ng-controller="MyController"
     ng-onboard-options="onboardOptions"
     ng-onboard-method="startOnboard" ...

Options in the controller 

$scope.onboardOptions = {
steps:[
{
    selector: '#step1',
    intro: "First tooltip"
},
{
    selector: '#step4',
    intro: "Second tooltip",
},
...

Call it either way 

<button ng-click="startOnboard();">Go</button>
$scope.startOnboard();
